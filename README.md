# Spotify-top10-s
Machine learning classification of Spotify songs using Python, Pandas and Scikit learn.

## Overview
Machine learning project predicting whether Spotify songs are highly popular using audio features.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Models
- Logistic Regression
- Decision Tree
- Random Forest

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Evaluation

## Results
Random Forest achieved the best performance.
